# Sanasto — Tekoäly työparina

| Käsite | Selitys |
| --- | --- |
| **Työparisykli** | Työtapa, jossa tekoäly tekee luonnoksen, ihminen muokkaa sen, tekoäly antaa rajattua palautetta ja ihminen tekee lopullisen päätöksen. |
| **Luonnos** | Ensimmäinen, keskeneräinen versio, jota ei vielä pidetä valmiina tuotoksena. |
| **Versio 1** | Muuttamattomana säilytetty lähtöversio, johon myöhempiä muutoksia verrataan. |
| **Versio 2** | Ihmisen tarkistama ja muokkaama versio ennen testilukijan palautetta. |
| **Testilukija** | Lukijan rooli, jonka avulla etsitään tekstistä mahdollisia jumiutumiskohtia. Tekoälyn simuloima testilukija ei korvaa oikeaa käyttäjätestiä. |
| **Päätösloki** | Kirjaus siitä, mikä ehdotus hyväksyttiin, mitä muokattiin tai hylättiin ja millä perusteella. |
| **Tarkistuslähde** | Todellinen aineisto, havainto tai asiantuntija, johon tuotoksen tärkeä väite verrataan. |
| **Ihmisen päätösvalta** | Vastuu määrittää tehtävä, arvioida ehdotukset ja hyväksyä lopputulos. |
