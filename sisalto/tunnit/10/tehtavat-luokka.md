# Opiskelutehtävät — Neutraali työkalukoe

> Tällä tunnilla on yksi ydintehtävä. Vertaat kahta käytettävissäsi olevaa tekoälypalvelua samalla tehtävällä. Jos käytössäsi on vain yksi palvelu, vertaa kahta eri asetusta tai tee koe opettajan antamilla anonyymeillä vastauksilla.

## Tehtävä 10.1 — Sama tehtävä, samat ehdot — suositeltu

### 1. Määritä tehtävä

Valitse tehtävä, jonka laatua osaat arvioida. Käytä vain julkista, itse tehtyä tai harjoitusta varten annettua aineistoa.

Kirjoita:

- tehtävä ja käyttötarkoitus
- käyttäjä tai kohderyhmä
- aineisto
- hyväksyttävä lopputulos

### 2. Tarkista porttiehdot

| Porttiehto | Työkalu A | Työkalu B |
| --- | --- | --- |
| Saako aineistoa käyttää tässä palvelussa? |  |  |
| Onko tarvittava ominaisuus käytettävissä? |  |  |
| Mahtuuko aineisto käsittelyyn? |  |  |
| Onko käyttöoikeus tai lisenssi kunnossa? |  |  |

Jos jokin pakollinen ehto ei täyty, älä syötä aineistoa palveluun. Kirjaa hylkäyksen syy.

### 3. Päätä arviointikriteerit etukäteen

Valitse 3–5 kriteeriä. Kuvaa, mistä hyvä tulos tunnistetaan.

| Kriteeri | Hyvän tuloksen tuntomerkki |
| --- | --- |
|  |  |
|  |  |
|  |  |

### 4. Aja koe

Käytä molemmissa palveluissa:

- samaa promptia
- samaa aineistoa
- uutta keskustelua
- samaa tavoiteltua vastausmuotoa

Kirjaa palvelun tai käyttöympäristön nimi sekä näkyvissä oleva malli tai asetus. Tallenna vastaukset kokonaisina.

### 5. Arvioi näyttö

Jos mahdollista, nimeä vastaukset ensin A:ksi ja B:ksi niin, ettei arvioija tiedä, mistä palvelusta ne tulivat.

| Kriteeri | Vastaus A: havainto | Vastaus B: havainto | Kumpi täytti kriteerin paremmin? |
| --- | --- | --- | --- |
|  |  |  |  |
|  |  |  |  |
|  |  |  |  |

Kirjaa lisäksi, paljonko kumpikin vaati jälkikorjausta ja mikä käytännön vaihe oli hankalin.

### 6. Tee rajattu valinta

Kirjoita 5–7 virkkeen päätös:

- minkä työkalun valitset tähän tehtävään
- mitkä kaksi havaintoa ratkaisevat valinnan
- mitkä porttiehdot täyttyivät
- mikä vertailussa jäi epävarmaksi
- milloin koe pitäisi uusia

Älä kirjoita, että valittu työkalu on yleisesti paras. Päätelmä koskee vain testaamaasi tehtävää ja käyttöympäristöä.

### Tekoälyvaihe — etsi perustelun aukko

```text
Tässä ovat ennalta määritellyt kriteerini, havaintotaulukkoni ja
johtopäätökseni. Esitä kolme kysymystä, jotka paljastavat, jos
johtopäätös ei seuraa havainnoista. Älä valitse työkalua puolestani
äläkä arvioi vastausten tyyliä ilman nimeämääni kriteeriä.
```

Tee kysymysten perusteella yksi nimetty korjaus päätökseesi tai kerro, miksi päätös ei muuttunut.

---

**Tallennettava tuotos:** testisuunnitelma, porttiehdot, molemmat vastaukset, havaintotaulukko, tehtäväkohtainen valinta ja yksi palautteen perusteella tehty tarkennus.
