# Opiskelutehtävät — Konteksti ratkaisee

> **Tällä tunnilla on vain yksi tehtävä**, mutta se harjoittaa yhtä koko kurssin tärkeimmistä taidoista. Kontekstin ja promptin hallinta erottaa tekoälyn satunnaiskäyttäjän vastuullisesta käyttäjästä. Tee tehtävä huolella.

## Tehtävä 4.1 — Huono ja hyvä prompti rinnakkain — SUOSITELTU

**Tavoite:** Näet omin silmin, miten dramaattisesti hyvä konteksti ja terävä prompti muuttavat tekoälyn vastausta. Kokeilet samaa ongelmaa kahdessa muodossa — laihana ja kunnolla rakennettuna — ja analysoit eroja.

### Valitse ongelmasi

Valitse yksi kysymys, joka **oikeasti kiinnostaa sinua tai liittyy arkeesi, opiskeluusi, harrastukseesi tai työhön** (voit myös kuvitella työtilanteen). Esimerkkejä eri aiheista:

- "Miten kirjoitan hyvän esitelmän rungon?"
- "Miten suunnittelen viikon ruokalistan?"
- "Miten valmistaudun kuljettajantutkintoon?"
- "Miten teen treeniohjelman juoksun aloittamiseen?"
- "Miten parannan englannin esseen rakennetta?"
- "Miten valmistaudun palaveriin tai haastatteluun?"

Mitä konkreettisempi oma tarpeesi, sitä parempi havainto syntyy.

### Vaiheet

#### Vaihe 1 — Huono prompti: paljas kysymys

Avaa ChatGPT, Claude tai Copilot. Kysy valitsemasi kysymys **ilman mitään kontekstia** — kuten kysyisit hakukoneelta.

> Esim. "Miten suunnittelen viikon ruokalistan?"

Tallenna vastaus (kopioi tai ota kuvakaappaus).

#### Vaihe 2 — Hyvä prompti: rakenna konteksti

Avaa **uusi keskustelu**, jotta edellinen vastaus ei vaikuta tulokseen. Rakenna prompti viidestä osasta:

| Osa | Mitä sisältö? |
|---|---|
| **Rooli** | Kuka olet? Esim. *"Kirjoitan ensimmäistä esitelmääni, enkä ole tehnyt tällaista aiemmin."* |
| **Tausta** | Mitä olet jo tehnyt? Esim. *"Olen kerännyt muutaman lähteen, mutta minulla ei ole rakennetta."* |
| **Tavoite** | Mitä haluat? Esim. *"Haluan oppia jäsentämään aiheen itse, en valmista tekstiä."* |
| **Rajaukset** | Mitä et halua? Esim. *"En halua vaikeita termejä — selitä yksinkertaisesti."* |
| **Formaatti** | Miten vastataan? Esim. *"Anna 3–5 askelta listana, ei pitkää esseetä."* |

Kirjoita prompti yhteen kappaleeseen tai listana — kunhan kaikki viisi osaa ovat mukana. Lähetä se mallille ja tallenna vastaus.

#### Vaihe 3 — Iteroi: paranna vastausta vielä kerran

Hyväkään ensimmäinen vastaus ei ole vielä loppu. Kokeneet käyttäjät eivät anna kerralla "kaikkea oikein" -promptia — he **tarkentavat kierros kierrokselta**. Kirjoita vielä yksi viesti, jolla viilataan vastausta. Esimerkkejä:

- "Tarkenna kohta 3 konkreettisella esimerkillä."
- "Anna myös yksi esimerkki, jossa kohta menee tarkoituksella pieleen, jotta opin tunnistamaan virheen."
- "Selitä tämä niin, että ymmärtäisin sen, vaikka en olisi koskaan tehnyt tällaista aiemmin."

Tallenna myös tämä vastaus. Tämä on iteratiivisuuden ydin — tekoäly ei ole arpapeli, jossa joko osuu tai ei. Se on keskustelu, jossa parannat ohjausta sitä mukaa kun näet, mihin se menee.

#### Vaihe 4 — Vertaa vastauksia

Aseta kolme vastausta (huono, hyvä, iteroitu) vierekkäin ja täytä taulukko:

| Mitä vertaillaan | Huono | Hyvä | Iteroitu |
|---|---|---|---|
| Pituus (sanaa, suurin piirtein) | | | |
| Sopivuus tasollesi (liian helppo / sopiva / liian vaikea) | | | |
| Osuvuus juuri sinun ongelmaan | | | |
| Käytännöllisyys (voinko soveltaa heti?) | | | |
| Tarkkuus (yleinen vai juuri sinulle) | | | |

#### Vaihe 5 — Kirjoita lyhyt pohdinta

Vastaa 4–6 lauseella:

- Mikä oli huonon ja hyvän version välillä suurin ero?
- Mikä viidestä kontekstin osasta (rooli, tausta, tavoite, rajaukset, formaatti) toi mielestäsi suurimman parannuksen?
- Mitä iterointi (vaihe 3) muutti — ja mitä tämä kertoo siitä, miten tekoälyä kannattaa käyttää jatkossa?
- Yhden lauseen sääntö, jonka otat tästä mukaan: *"Jatkossa annan tekoälylle aina…"*

> **Vinkki:** Tehtävää ei palauteta, mutta säilytä muistiinpanodokumenttisi. Vaiheessa 5 kirjoittamasi *"Jatkossa annan tekoälylle aina…"* -sääntö on yksi tärkeimmistä asioista, jotka viet koko kurssilta mukanasi.

**Konteksti ratkaisee -tunnin jälkeen osaat antaa tekoälylle kontekstin, joka tuottaa käyttökelpoisia vastauksia.**
