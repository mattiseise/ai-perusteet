# Sanasto — Promptikortti

| Käsite | Selitys |
| --- | --- |
| **Promptikortti** | Uudelleen käytettävä prompti sekä sen käyttötilanne, syötteen paikka, laatukriteerit, testihavainnot ja tunnetut rajat. |
| **Pysyvä ohje** | Promptin osa, joka säilyy käyttökerrasta toiseen. |
| **Vaihtuva syöte** | Aineisto tai tieto, joka vaihdetaan uutta käyttökertaa varten. |
| **Laatukriteeri** | Ennen ajoa päätetty havaittava ominaisuus, jolla vastausta arvioidaan. |
| **Hallittu muutos** | Menettely, jossa versioon 2 muutetaan vain yksi olennainen asia ja muut ehdot pidetään samoina. |
| **Versiohistoria** | Kirjaus promptin versioista, muutoksista ja niiden havaituista vaikutuksista. |
| **Tunnettu raja** | Tilanne tai aineisto, jossa promptikorttia ei ole syytä käyttää sellaisenaan. |
| **Siirrettävyys** | Se, kuinka hyvin kortti toimii erilaisilla mutta samaan tehtävään kuuluvilla syötteillä. |
