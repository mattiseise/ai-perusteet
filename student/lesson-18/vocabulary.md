# Lesson 18: Sanasto (OSP2:n yhteenveto)

| Termi | Määritelmä |
|-------|-----------|
| **Promptaus** (prompting) | Tekoälylle esitetty pyyntö tai käsky |
| **Konteksti** | Tieto, jonka annat tekoälylle, jotta se ymmärtää, mitä haluat |
| **Hallusinaatio** | Tekoälyn tuottama väärä tieto, joka kuulostaa totuudelliselta |
| **Chat-pohjainen assistentti** | Tekoäly, jota käyttää kirjoitetuilla pyynnöillä (ChatGPT, Claude jne.) |
| **Generatiivinen malli** | Tekoäly, joka luo uutta sisältöä (tekstiä, kuvaa, koodia) |
| **Koodigenerointi** | Tekoälyn avulla kokonaisten ohjelmien tai funktioiden tuottaminen |
| **Testaus** | Koodin ajaminen tunnetuilla syötteillä ja tulosten tarkistaminen |
| **Debuggaus** | Virheiden etsintä ja korjaaminen koodissa |
| **Dokumentaatio** | Kirjallinen selitys siitä, mitä koodi tekee ja miten sitä käytetään |
| **Feature flag** | Ohjelmistokehityksessä työkalu, jonka avulla ominaisuuksia voidaan ottaa pois päältä ilman uudelleenkäynnistystä |
