# Lesson 17: Sanasto

| Termi | Määritelmä |
|-------|-----------|
| **Debuggaus** (debugging) | Prosessi, jossa etsit ja korjaat koodissa olevia virheitä (bugeja) |
| **Testaus** (testing) | Koodin ajaminen tunnetuilla syötteillä ja tulosten tarkistaminen |
| **Verifiointi** (verification) | Prosessi, jossa varmistetaan, että koodi tekee oikein sen, mitä sen pitäisi tehdä |
| **Refaktorointi** (refactoring) | Koodin muuttaminen paremmaksi (luettavampi, nopeampi) ilman toiminnon muuttamista |
| **Koodin tarkistus** (code review) | Toisen ihmisen tai prosessin avulla koodin laadun ja turvallisuuden tarkistaminen |
| **Haavoittuvuus** (vulnerability) | Kodin tai järjestelmän heikkous, jota hyökkääjä voi hyödyntää |
| **SQL-injektio** | Tietoturvahyökkäy, jossa hyökkääjä syöttää SQL-koodia ja pääsee käsiksi tietoihin |
| **Virheviesti** | Tekstiviesti, joka kertoo, mitä meni pieleen, kun koodi ajettiin |
