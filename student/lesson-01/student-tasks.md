# Opiskelutehtävät

## Tehtävä 1.1: AI vai ei AI — Luokitteluharjoitus

### Tavoite
Harjoitella AI:n ja muun ohjelmoinnin erottamista tosielämän esimerkkien perusteella.

### Ohjeet (ryhmätehtävä, 2–3 henkilöä)

Jaa ryhmälle alla oleva taulukko. Kunkin esimerkin kohdalla täytä sarake "AI / Ei AI / Epäselvä" ja perustele päätös. Jos vastaus on "Epäselvä", selitä, miksi se on epäselvä.

| Järjestelmä | Kuvaus | AI / Ei AI / Epäselvä | Perustelu |
|---|---|---|---|
| Sähköpostin spämmi-suodatin | Automaattisesti lajittelee viestit roskapostiin sanojen perusteella | | |
| Netflix-suositukset | Näyttää elokuvia ja sarjoja, jotka sinulla on todennäköisesti mieltä | | |
| Pankin lainahakemuksen hyväksyntä | Ottaa päätöksen hakemuksen hyväksymisestä tai hylkäämisestä automaattisesti | | |
| Älypuhelimen puhelinluettelo | Järjestää yhteystiedot aakkosellisesti | | |
| Google Maps -navigaattori | Ohjaa sinut reiteille reaaliaikaisesti liikenteen perusteella | | |
| ATM-automaatti | Antaa rahaa, kun syötät PIN-koodin | | |
| Hiiren auto-correct | Korvaa väärän kirjoituksen oikealla sanalla | | |

### Odotettu tuotos

Täytetty taulukko, jossa jokainen rivi on perusteltu. Ryhmä laatii yhteenvedon: "Mitkä kriteerit auttavat erottamaan AI:n muusta ohjelmoinaista?"

**Jos teet tehtävän yksin:**
Valitse 3 esimerkkiä ja täytä ne täysin. Kirjoita lopuksi lyhyt vastaus: "Mikä on tärkein ero AI:n ja tavallisen ohjelmoinnin välillä tämän harjoituksen perusteella?"

## Tehtävä 1.2: Petoksentunnistus — Sääntö vs. Tekoäly

### Tavoite
Ymmärtää, miksi tekoäly on tehokkaampi kuin kiinteät säännöt monimutkaisissa päätöksentekotilanteissa.

### Ohjeet (ryhmätehtävä, 2–3 henkilöä)

Pankin tehtävä: suojata asiakkaiden tilit petoksilta. Analyysi 10 transaktiota alla olevan datan perusteella.

| # | Summa (€) | Kello | Paikka | Asiakkaan tavallinen käyttö | Lopputulos |
|---|---|---|---|---|---|
| 1 | 5 500 | 14:30 | Koulu | Yleensä < 1 000 | Laillinen |
| 2 | 2 000 | 03:15 | Tuntematon maa | Harvoin yöllä | **Petos** |
| 3 | 150 | 10:00 | Ruokakauppa | Päivittäinen | Laillinen |
| 4 | 8 000 | 19:45 | Koti | Kerran kuussa | Laillinen |
| 5 | 300 | 02:30 | Verkkokauppa | Ei koskaan yöllä | **Petos** |
| 6 | 50 | 12:00 | Kahvila | Päivittäinen | Laillinen |
| 7 | 10 000 | 15:00 | Tuntematon maa | Ensimmäinen kerta | **Petos** |
| 8 | 3 000 | 20:00 | Koti | Kuukausittain | Laillinen |
| 9 | 600 | 23:45 | Tuntematon kauppa | Ei koskaan | **Petos** |
| 10 | 75 | 11:00 | Ruokakauppa | Päivittäinen | Laillinen |

**Vaihe 1:** Ryhmä kehittää sääntöpohjaisen suodattimen. Kirjoita säännöt: "Jos [ehto], merkitse petokseksi."

**Vaihe 2:** Testaa sääntösi kaikkia 10 transaktiota vastaan. Kuinka monta petosta huomaat oikein? Kuinka monta väärää hälytystä?

**Vaihe 3:** Pohdi: Mikä tekee tämän ongelman vaikeaksi kiinteille säännöille? Mitä tekoäly voisi tehdä paremmin?

### Odotettu tuotos

Dokumentti, joka sisältää:
- Ryhmän kehittämät säännöt
- Testien tulokset (oikeat positiiviset, väärät positiiviset)
- Kirjallinen vastaus: "Miksi tekoäly olisi parempi tähän kuin säännöt?" (3-5 lausetta)

**Jos teet tehtävän yksin:**
Tee vaiheet 1–2. Kirjoita vaiheelle 3 lyhyt vastaus.

## Tehtävä 1.3: Omat sovellukset — Missä käyttää tekoälyä?

### Tavoite
Soveltaa oppeja omiin päivittäisiin sovelluksiin ja palveluihin.

### Ohjeet (pareittain tai yksin)

Valitse 5 sovellusta tai verkkopalvelua, jota käytät säännöllisesti (esim. Instagram, Snapchat, Discord, YouTube, Amazon, Spotify, Google Maps, pelit).

Täytä taulukko jokaiselle:

| Sovellus | Funktio, joka voi käyttää AI:ta | Käyttyykö todennäköisesti AI? | Perustelu |
|---|---|---|---|
| | | Kyllä / Ei / En tiedä | |

### Odotettu tuotos

Täytetty taulukko. Kirjoita lyhyt pohdiskelu: "Missä tekoäly on liukenetusti osa omaa digitaalista elämääsi?"

**Jos teet tehtävän yksin:**
Valitse 3 sovellusta ja täytä taulukko. Kirjoita pohdinnoista lyhyt kappale.
