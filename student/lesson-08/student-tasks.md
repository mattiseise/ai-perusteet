# Opiskelutehtävät: Etiikka, tekijänoikeudet ja ympäristövaikutukset

## Tehtävä 1: Tekijänoikeudet ja koulutusdata — case-analyysi

### Tavoite
Ymmärtää tekijänoikeusten ja koulutus datan eettisiä ja oikeudellisia näkökohtia.

### Ohjeet

1. **Tutustu tositapaukseen:**
   Valitse yksi näistä oikeustapauksista:
   - **OpenAI vs. kirjailijoiden yhdistys** (2023) — kirjailijoiden korvausvaatimus ChatGPT:n koulutuksesta
   - **Stability AI vs. kuvataiteilijat** (2023) — Midjourneyn ja Stable Diffusionin koulutus kuvataiteilijoiden ilman lupaa
   - **Google vs. kirjoittajat** — Googlen koulutus data hankinnan laatu legaliteetti

2. **Tutkimukset tapauksen:**
   - Mikä on kantaja ja vastapuoli?
   - Mitä väitettä kantaja nostaa?
   - Mitä väitettä vastapuoli esittää?
   - Mikä on "reilu käyttö" (fair use) -argumentti?

3. **Täytä analyysi taulukko:**

   | Näkökulma | Kantajan näkemys | Vastapuolen näkemys | Sinun näkemys |
   |---|---|---|---|
   | Onko tekijöille pyydetty lupa? | | | |
   | Pitäisikö tekijöille maksaa korvaus? | | | |
   | Onko koulutus "transformatiivista"? | | | |
   | Mitkä ovat oikeudelliset ja eettiset näkökohdat? | | | |

4. **Kirjoita johtopäätös (4–5 lausetta):**
   - Mikä puoli on omasta mielestä oikeutuneempi?
   - Mitä ammattilaisena pitäisit tekijänoikeusvastuu mielessä käyttäessäsi tekoälyä?

## Tehtävä 2: Algoritmisen harhan analyysi

### Tavoite
Oppia tunnistamaan ja analysoimaan kuinka algoritminen harha syntyy ja miten sitä voi testata.

### Ohjeet

**Valitse yksi tunnettu algoritmiharha-tapaus:**
- **Amazon:n rekrytointialgoritmi** — yritti systemaattisesti poistaa naisia
- **COMPAS-algoritmi** — oikeus järjestelmän uusintariskiarviointi oli puolueellinen rodullisuuden suhteen
- **Kasvojen tunnistus** — toimii huonommin tummaihoisille
- **Lainahyväksyntä** — algoritmi diskriminoi tiettyjen etnisten ryhmien perusteella

1. **Etsi tapaus ja dokumentoi:**
   - Mitä algoritmi teki?
   - Kenen kohtaan se oli puolueellinen?
   - Miksi harha syntyi? (Minkä koulutus datan vuoksi?)

2. **Analysoi pääkohdat:**

   | Kohta | Kuvaus |
   |---|---|
   | Algoritmin tehtävä | Mitä sillä yritettiin tehdä? |
   | Harhasta kärsivä ryhmä | Ketä se diskriminoi? |
   | Harhan lähde | Minkä datan vuoksi harha syntyi? |
   | Seuraukset | Mitä vahinkoa se aiheutti? |
   | Ratkaisu | Miten sitä korjattiin tai lopetettiin? |

3. **Pohdi sopeutuvuutta:**
   - Jos sinua pyydettäisiin testaamaan samanlaista algoritmia organisaatiossasi, mitä testejä ajaisit harhan varalta?
   - Kuka olisi vastuussa, jos algoritmi osoittautuu puolueelliseksi?

## Tehtävä 3: Datan merkitsijät ja globaali työ

### Tavoite
Ymmärtää, mitä "näkymätöntä työtä" AI:n opetukseen liittyy ja mikä sen eettinen merkitys on.

### Ohjeet

1. **Tutkia datan merkitsijöiden olosuhteita:**
   - Etsi artikkeleita tai dokumentaareita aiheesta "AI data labeling" tai "Amazon Mechanical Turk"
   - Tutki: mitä he tekevät, palkkaa, työolosuhteita, mielenterveysongelmia

2. **Dokumentoi löydöksesi:**

   | Olosuhde | Kuvaus | Lähde |
   |---|---|---|
   | Palkkaus | Kuinka paljon merkitsijät saavat? | |
   | Työn luonne | Mitä merkitsijät tekevät päivittäin? | |
   | Psyykkinen kuormitus | Mitkä haasteet heillä on? | |
   | Oikeudet | Onko heillä sopimuksia, vakuutusta? | |

3. **Pohdi vastuuta:**
   - Kun käytät ChatGPT:tä, käytät mallia, jonka opettivat nämä työntekijät.
   - Kuka on vastuussa heidän hyvinvoinnistaan? (OpenAI? Sinä? Molemmat?)
   - Pitäisikö merkitsijöille maksaa enemmän?
   - Pitäisikö hänen saada pääsy malleihin, joita he auttivat rakentaa?

4. **Kirjoita eettinen kannanotto (3–4 lausetta):**
   - Millä tavoin ammattilaisesti ottaisit nämä näkökohdat huomioon käyttäessäsi tekoälyä?

## Tehtävä 4: Ympäristövaikutus ja energia

### Tavoite
Ymmärtää tekoälyn ympäristöjalanijälkeä ja harkinnan tarpeellisuutta käytössä.

### Ohjeet

1. **Etsi tietoja ChatGPT:n (tai muiden mallien) energiankulutuksesta:**
   - Kuinka paljon sähköä yksi query kuluttaa?
   - Kuinka paljon vettä data-keskukset käyttävät?
   - Mitä kasvihuonekaasuja generoi?

2. **Laske ja vertaa:**

   | Metriikka | Luku | Vertailukohta |
   |---|---|---|
   | Sähkö per query | X wattituntia | = vertaa tavallisen LED-lampun käyttöön tunti |
   | Vesi vuodessa | X miljoonaa gallonaa | = vertaa kaupungin vesijohto kulutukseen |
   | Hiilijalanjälki | X tonnia CO2/vuosi | = vertaa autojen tai lentoliikenteen päästöihin |

3. **Pohdi rationaalista käyttöä:**
   - Jos organisaatiosi käyttäisi tekoälyä jokaiseen tehtävään (esim. jokainen email), kuinka paljon se kasvaisi energiakulutusta?
   - Mitkä tehtävät ovat "arvoisia" tekoälyn käytön energiakustannukselle?
   - Mitä tarkoittaisi "harkittu" tekoälyn käyttö organisaatiossasi?

## Tehtävä 5: Professionaalinen eettinen kannanotto

### Tavoite
Kehittää kykyä reflektoida omaa ammattilista vastuuta tekoälyn kanssa.

### Ohjeet

Kirjoita **lyhyt eettinen kannanotto** (1–2 sivua):

**Aiheiden joukosta valitse yksi:**
1. Miten ottaisit tekijänoikeudet huomioon käyttäessäsi generatiivisia AI:ta ammattisissa projekteissa?
2. Miten varmistaisiosi, etteivät käyttämäsi algoritmit ole puolueellisia?
3. Pitäisikö organisaatiossasi olla "tekoälyn eettinen neuvosto" — ja mitä se tekisi?
4. Voiko tekoälyn käyttö olla eettisesti kestävää ilman suurempia rakenteellisia muutoksia (esim. tekijöille maksaminen)?

**Esitysmuoto:**
- Johdanto: Mikä on ongelmasi?
- Argumentit: Mitä ajattelisi puoltaa valintaasi?
- Vastalauseet: Mitä muut voivat vastustaa?
- Johtopäätös: Mikä on ammatillinen kantasi?

---

## Ohjeet tehtäviin

- **Tehtävät 1–4** voidaan tehdä yksin tai pareittain.
- **Tehtävä 5** on reflektiivinen ja kannattaa tehdä yksin, mutta voit käsitellä tuloksia pareittain.
- Käytä luotettavia lähteitä (tieteelliset artikkelit, uutispalvelut, tutkimusraportti).
